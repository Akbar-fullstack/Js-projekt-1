# Js-projekt-1
В этом проекте я реализовал базовый калькулятор, который выполняет простые арифметические операции (сложение, вычитание, умножение, деление).  Особенности:  Ввод данных через prompt  Логика выбора операций с помощью switch case  Кнопки для запуска и управления вычислениями  Простая и понятная структура кода  Проект создан как учебная практика
